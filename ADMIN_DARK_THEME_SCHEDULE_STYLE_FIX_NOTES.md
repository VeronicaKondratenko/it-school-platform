# Admin dark theme schedule/style fix

Виправлено саме темну тему адмін-панелі.

## Що змінено
- У `frontend/admin-dashboard.html` додано окремий блок CSS для `html[data-theme="dark"] body.admin-dashboard-page`.
- Розділи `Огляд`, `Користувачі`, `Курси`, `Групи`, `Розклад`, `Звіти`, `Сповіщення` мають використовувати однакові темні картки, таблиці, поля вводу й кольори тексту.
- `Розклад` більше не має виглядати як чужий блок відносно решти адмінки в темній темі.
- Світлу тему не відкотили: для неї залишені попередні правила читабельності.

## Оновлення
Замінити файл `frontend/admin-dashboard.html`, виконати `git add`, `commit`, `push`, потім оновити frontend на Render і зробити `Ctrl + F5`.
