# Виправлення лічильників курсів викладача

## Проблема
На сторінці викладача у вкладці `Мої курси` курси відображались, але біля кожного курсу було:

- `0 студентів`
- `0 груп`
- `0 завдань`

Причина: endpoint `/api/teacher/courses` повертав тільки `id`, `title`, `description`, а frontend очікував додаткові поля `students_count`, `groups_count`, `assignments_count`. Через відсутність цих полів frontend показував fallback-значення `0`.

## Що змінено
Оновлено файл:

- `backend/app/api/endpoints/teacher.py`

Тепер `/api/teacher/courses` повертає для кожного курсу:

- `groups_count` — кількість груп викладача, прив'язаних до курсу;
- `students_count` — кількість унікальних студентів у цих групах;
- `assignments_count` — кількість завдань курсу через реальний зв'язок `Course -> Discipline -> Topic -> Assignment`.

Дисципліни залишаються технічним внутрішнім шаром у backend, але не показуються користувачу в UI.

## Як перевірити
1. Оновити backend на Render.
2. Зайти як викладач.
3. Відкрити `Мої курси`.
4. Біля курсів мають бути реальні кількості студентів, груп і завдань, а не нулі.
