# Final light theme UI/readability fix

Зміни v15:

1. Виправлено білий/невидимий текст у світлій темі на сторінці звітів адміністратора.
   - Кнопки звітів (`btn-sm`, `btn-edit`) тепер мають темний текст на світлому фоні.
   - Таблиці звітів і динамічні inline-блоки мають примусовий читабельний колір.

2. Сторінки звернень у світлій темі приведено до стилю всієї системи.
   - Sidebar на сторінках `admin-questions.html`, `teacher-questions.html`, `questions.html` тепер залишається темно-синім, як на інших внутрішніх сторінках.
   - Активні пункти меню, логотип, роль і профіль у sidebar мають правильні кольори.

3. Виправлено обрізання тексту в таблиці звернень адміністратора.
   - Таблиця більше не ховає текст за горизонтальним скролом.
   - Колонки мають фіксовані відсоткові ширини, текст переноситься рядками.
   - Статуси й довгі теми видно повністю.

4. Пункт “Дисципліни” вже не має показуватися у фінальному UI адміністратора. Дисципліни залишаються лише як технічний шар backend, щоб не ламати зв’язки тем/завдань.

Після оновлення Render потрібно зробити Ctrl+F5 у браузері.
