# Тестові акаунти (після `seed_realistic`)

Єдиний перелік акаунтів для входу після сидингу бази. Паролі тестові — лише для
локальної розробки. **У production створюйте власні акаунти й змінюйте паролі.**

| Роль     | Email                  | Пароль     |
|----------|------------------------|------------|
| Адмін    | admin@school.com       | admin123   |
| Викладач | teacher@example.com    | password   |
| Студент  | student@example.com    | password   |

Додаткові згенеровані акаунти:

- Викладачі: `teacher1@example.com`, `teacher2@example.com`, `teacher3@example.com` — пароль `password`
- Студенти: `1student@example.com` … `20student@example.com` — пароль `password`

> Усі паролі демо-даних = `password` (окрім адміна). Це навмисно для зручності
> перевірки. Публічна реєстрація вимагає пароль від 8 символів і завжди створює
> роль «студент».
